Near Node Flash documentation!
===================================

**Near Node Flash** is a Kubernetes based storage orchestration service that accepts API requests and allocates and configures file systems.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
